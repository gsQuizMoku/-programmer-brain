# :speech_balloon:進め方（仮）
1. 目標時間の前半は問題文のみで考える
2. 目標時間の後半はヒントも見ながら考える
3. 目標時間が過ぎたら提出する
4. 考え方を見ながらみんなで答え合わせ
5. 回答を修正する
